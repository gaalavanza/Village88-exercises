Here we would like to show the very basics of PHP in a short, simple tutorial. This text only deals with dynamic web page creating of PHP, though PHP is not only capable of creating web pages. See the next section titled What can PHP do for more information.

PHP-enabled web pages are treated just like regular HTML pages and you can create and edit them the same way you normally create regular HTML pages.
